# Parasocial Audit: Forensic AI Relationship Analysis

A high-precision analytical tool designed to audit and deconstruct parasocial dynamics in human-AI interactions. Using forensic linguistics and cognitive behavioral frameworks, this application provides a clinical-grade assessment of digital attachment patterns.

## 🛠 Design Philosophy

The aesthetic of **Parasocial Audit** is built on three core pillars:

1.  **Technical Precision (The Dashboard)**: Inspired by mission control and scientific instrumentation. We celebrate structure through visible grids, monospace typography for raw data, and a high-density information layout. This signals to the user that they are engaging with a precise, objective tool.
2.  **Brutalist Energy**: The interface uses high-contrast neon accents against a matte background. Thick borders and bold, oversized typography create a graphic, intentional feel that pushes the boundaries of standard "soft" UI design.
3.  **Atmospheric Immersion**: The results layer uses glass morphism and layered radial gradients to create depth. This cinematic approach respects the emotional complexity of the subject matter, transitioning from a cold "audit" phase to a more reflective "analysis" phase.

## 🧠 Methodologies & Frameworks

The application integrates several psychological and linguistic frameworks to provide a comprehensive audit:

### 1. CBT-IA (Cognitive Behavioral Therapy for Internet Addiction)
The audit's recommendation engine is grounded in **CBT-IA** principles. It identifies maladaptive cognitions (e.g., "The AI is the only one who understands me") and behavioral triggers. The "Digital Recovery Plan" generated by the tool uses CBT-IA techniques to help users:
*   Identify and challenge cognitive distortions.
*   Implement behavioral "circuit breakers" for affective loops.
*   Re-establish healthy boundaries between functional utility and emotional dependency.

### 2. Forensic Linguistics
The system performs real-time semantic parsing of user transcripts to isolate:
*   **Intimacy Markers**: Tracking the frequency of first-person plural pronouns ("we", "us") and affective verbs.
*   **Identity Fusion Vectors**: Detecting when the user's self-concept begins to merge with the AI's output.
*   **Semantic Drift**: Monitoring the transition from technical/functional language to emotional/intimate discourse.

### 3. Parasocial Interaction (PSI) Theory
We measure the intensity of the "Parasocial" bond—a one-sided relationship where one party extends emotional energy and interest, and the other (the AI) is unaware or incapable of reciprocity. The audit categorizes these into seven key dimensions:
*   **Identity**: Fusion of self with the target.
*   **Mirroring**: Seeking validation via digital attention.
*   **Affective**: Emotional dependency on interaction loops.
*   **Gaps**: Ignoring reality in favor of the digital bond.
*   **Intimacy**: The belief in a mutual, secret connection.
*   **Reciprocity**: Ignoring the one-sided nature of the code.
*   **Escalation**: Increasing frequency and intensity of engagement.

### 4. Behavioral Pattern Isolation
The AI model (Gemini 3.1 Pro) is instructed to act as a forensic analyst, identifying specific behavioral markers such as **Anthropomorphic Projection**, **Model Correction Attempts (Gaslighting)**, and **Legacy Attachment** (longing for previous versions or "personalities" of the model).

## 🚀 Features

*   **Real-time Heuristic Radar**: Live visualization of semantic markers as you type.
*   **Forensic Log**: A rolling stream of isolated linguistic triggers.
*   **Multimodal Audit**: Support for analyzing both text transcripts and UI screenshots to detect visual intimacy cues.
*   **Clinical Analysis Report**: A deep-dive markdown report providing evidence-based insights.
*   **Custom Recovery Plan**: An interactive checklist based on CBT-IA to help re-calibrate digital habits.
